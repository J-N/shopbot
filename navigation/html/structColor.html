wander.c

This is a more advanced version of the tracker program. Makes the Create wander around, trying to follow the floor and not hit anything. It will do this until it finds the color it is supposed to track, at which point it will move towards the object of that color. If the robot loses the object, it will begin wandering around again. Requires OpenCV.
